# Welcome to AI Chat Studio Storybook

This is the component documentation for **AI Chat Studio**, a modern AI chat interface built with React, TypeScript, and Tailwind CSS.

## What is Storybook?

Storybook is a frontend workshop for building UI components and pages in isolation. It helps you develop and share hard-to-reach states and edge cases without needing to run your whole app.

## Quick Start

### Running Storybook

```bash
# Install dependencies
pnpm install

# Start Storybook
pnpm run storybook
```

Storybook will start on `http://localhost:6006`

### Building Static Storybook

```bash
# Build static Storybook site
pnpm run build-storybook
```

## Design System

### Colors
- **Primary**: Purple gradient (`#667eea` to `#764ba2`)
- **Secondary**: Blue tones
- **Success**: Green
- **Error**: Red
- **Warning**: Yellow
- **Info**: Blue

### Typography
- **Font Family**: Inter, system fonts
- **Sizes**: xs (12px), sm (14px), base (16px), lg (18px), xl (20px), 2xl (24px)

### Spacing
- Uses Tailwind's default spacing scale (0.25rem increments)
- Common values: 1 (4px), 2 (8px), 4 (16px), 6 (24px), 8 (32px)

## Usage Guidelines

### Component Props
All components are fully typed with TypeScript. Use the Controls panel to experiment with different prop values.

### Accessibility
All components follow WCAG 2.1 AA standards. Use the Accessibility addon to check for issues.

### Responsive Design
Test components at different viewport sizes using the viewport toolbar:
- Mobile: 375px
- Tablet: 768px
- Desktop: 1440px

### Internationalization
Switch languages using the locale toolbar to preview translated content.
